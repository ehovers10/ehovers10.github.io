Properties of propositions

  • Just as in categorical logic, we can use our semantics for propositional logic to define properties of certain propositions.
    • Tautology
      • A proposition is tautologous, if it is impossible for it to be false.
    • Contradiction
      • A proposition is contradictory, if it is impossible for it to be true.
    • Equivalence
      • Two propositions are equivalent, if they are true in all the same situations and false in all the same situations.

Properties and truth tables

  • We can determine what property (if any) that a particular WFF has by looking to its truth table.
    • Tautology
      • A proposition is tautologous if its column in the truth table has a T in every row.
      • This shows that it is true in every possible situation.
    • Contradiction
      • A proposition is contradictory if its column has an F in every row.
      • This shows that it is false in every possible situation.
    • Equivalence
      • Two propositions are equivalent, if their columns have the same truth value in every row.
      • This shows that for every possible situation, the two propositions agree.

Truth tables for arguments

  • Arguments aren't any different from complex propositions, as far as PL is concerned.
  • So, we can create truth tables for arguments the same way we did for complex propositions
    • First, create a column for every propositional variable that occurs in the argument.
    • Then create a column for every premise in the argument.
    • And finish with a column for the conclusion.
    • Just to make it clear that this is an argument, add indicators in each column to tell what it role it plays in the argument.
